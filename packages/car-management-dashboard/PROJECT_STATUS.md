## 🔄 系统工作流程

1. **用户访问** → 检查登录状态
2. **未登录** → 显示登录表单
3. **登录成功** → 获取租户列表 → 自动选择默认租户
4. **进入系统** → 基于当前租户显示数据
5. **租户切换** → 更新上下文 → 重新加载相关数据
